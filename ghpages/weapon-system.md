# Weapon System

Modular weapon and attachment system for Bevy.

## Features

- **Modular Weapons**: Easily define new weapon types.
- **Attachment System**: Dynamic attachments like scopes, lasers, and suppressors.
- **Weapon Builder**: Utility for programmatically constructing weapons.

## Components

- `Weapon`: Stores core stats (damage, fire rate).
- `AttachmentSlot`: Defines where attachments can be placed.
- `LaserPointer`: Specialized logic for laser visual effects.

## Usage

See `examples/weapon_demo.rs` for a basic setup.
