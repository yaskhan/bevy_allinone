# Getting Started

This guide will help you get up and running with the Bevy All-in-One project.

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (latest stable)
- [Bevy Engine](https://bevyengine.org/) dependencies for your OS

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yaskhan/bevy_allinone.git
   cd bevy_allinone
   ```

2. Run the project:
   ```bash
   cargo run --release
   ```
