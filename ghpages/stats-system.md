# Stats and Skills System

Comprehensive system for character progression and attribute management.

## Features

- **Dynamic Stats**: Health, Stamina, Mana, and custom attributes.
- **Skill Trees**: Progression paths with unlockable perks.
- **Experience (XP)**: Leveling logic and reward distribution.

## Components

- `Stats`: Core attributes container.
- `SkillContainer`: Tracks unlocked and active skills.
- `Experience`: Current XP and level data.

## Systems

- `stat_regeneration_system`: Handles passive recovery.
- `xp_gain_system`: Processes XP rewards from combat and quests.
- `skill_effect_system`: Applies modifiers based on active skills.
