# Quest and Dialog Systems

Tools for narrative-driven gameplay and NPC interaction.

## Features

- **Quest Graph**: Branching quest lines with multiple objectives.
- **Dynamic Dialog**: Tree-based dialog system with choices and consequences.
- **Journal**: Tracking active and completed tasks.

## Components

- `QuestGiver`: NPCs that can initiate quests.
- `DialogParticipant`: Entities capable of conversation.

## Systems

- `quest_tracking_system`: Monitors world events for quest progress.
- `dialog_ui_system`: Renders the conversation interface.
